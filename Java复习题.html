01
Java 应用程序与小程序之间有哪些区别？
Application：包含 main 方法，可独立运行，无需继承特定类。
Applet：无 main 方法，必须继承 Applet 类，无法独立运行，需要浏览器或 appletviewer 工具执行。
02
成员变量与局部变量的区别有哪些？
定义位置：成员变量定义在类中；局部变量定义在方法、代码块或形参列表中。
内存位置：成员变量存于堆 / 方法区；局部变量存于栈内存。
默认值：成员变量有系统默认值；局部变量无默认值，必须手动初始化。
作用域：成员变量作用域为整个类；局部变量仅在当前方法 / 代码块内有效。
03
什么是“多态”机制？Java 语言中是如何实现多态的？
多态：指同一个行为，作用在不同对象上会产生不同执行结果。
实现条件：存在继承关系、对子类方法进行重写、使用父类引用指向子类对象。
04
构造方法的作用？若类中没有声明构造方法，程序能正确执行吗？为什么？
构造方法作用：创建对象，同时初始化类的成员变量。
可以正常执行。原因：当类中没有手动定义构造方法时，编译器会自动生成一个默认无参构造方法。
05
什么是套接字？其作用是什么？
套接字：网络通信的端点，由 IP 地址 + 端口号组合而成。
作用：实现不同设备上应用程序之间的数据通信与传输。
06
什么是包？它的作用是什么？如何创建包？如何应用包中的类？
包：是管理类的命名空间，对应操作系统中的文件夹。
作用：分类管理类、避免类名重复、实现访问权限控制。
创建包：在 Java 源文件第一行编写代码 package 包名;
使用包中类：通过 import 语句导入，或直接书写完整包名 + 类名。
07
公共成员与私有成员有何区别？
公共成员（public）：任意位置都可访问，能够被子类继承。
私有成员（private）：仅当前类内部可以访问，子类、外部类均无法直接访问。
08
什么是字节码？采用字节码的最大好处是什么？
字节码：Java 源文件（.java）编译后生成的 .class 文件，是 JVM 可识别的中间指令。
最大好处：实现跨平台，做到一次编写，到处运行。
09
什么是流？Java 语言中分为哪两种流？这两种流有何差异？
流：数据传输的通道，用于程序读写数据。
字节流：以字节为单位读写，可处理图片、视频、文本等所有类型文件。
字符流：以字符为单位读写，仅处理纯文本文件，自带编码转换功能。
10
什么是平台无关性？Java 语言中是怎样实现平台无关性的？
平台无关性：同一套代码，无需修改、重新编译，即可在不同操作系统上运行。
实现方式：源码编译为与平台无关的字节码（.class）；不同系统安装对应版本 JVM，由 JVM 解析执行字节码，屏蔽系统差异。
11
静态变量与实例变量有哪些不同？
修饰符：静态变量用 static 修饰；实例变量无 static 修饰。
归属：静态变量属于类；实例变量属于对象。
共享性：静态变量被所有对象共享；实例变量为每个对象独有。
调用方式：静态变量通过类名调用；实例变量必须通过对象调用。
内存位置：静态变量存于方法区；实例变量存于堆内存。
12
Java 语言中定义的三个标准输入输出流是什么？它们对应什么设备？
System.in：标准输入流，对应设备为键盘。
System.out：标准输出流，对应设备为控制台。
System.err：标准错误输出流，对应设备为控制台。
13
什么情况下必须以类实现 Runnable 接口来创建线程？
Java 是单继承语言，当当前类已经继承了其他父类，无法再继承 Thread 类时，必须实现 Runnable 接口创建线程。
14
Java 语言中的网络编程分为哪几个层次？
传输层：Socket 编程，基于 TCP、UDP 协议，实现底层数据通信。
应用层：URL 编程，基于 HTTP 等协议，直接访问网络资源。
15
Java 语言程序有哪两种？它们的主类有哪些不同？
Java 应用程序（Application）：主类包含 main 方法，可独立运行，无需继承其他类。
Java 小程序（Applet）：主类无 main 方法，必须继承 Applet，不能独立运行。
16
Java 语言定义了哪些基本数据类型？各占几个字节？
共 8 种基本数据类型：byte(1), short(2), int(4), long(8), float(4), double(8), char(2), boolean(1字节处理)
17
为什么程序的多线程功能是必要的？Java 语言中实现多线程的方法有哪两种？
多线程的作用：提升 CPU 利用率、实现并发处理、提升程序响应速度。
两种实现方式：继承 Thread 类、实现 Runnable 接口。
18
写出至少 5 个 Java 语言中的常用包
java.lang, java.util, java.io, java.net, java.sql, javax.swing 等。
19
JDBC 中 DriverManager、Connection、Statement 和 ResultSet 接口的功能？
DriverManager：加载数据库驱动，获取数据库连接。
Connection：代表数据库连接会话，用于创建语句对象。
Statement：负责执行 SQL 语句。
ResultSet：封装查询返回的数据结果，提供数据遍历读取功能。
20
面向对象编程中，什么是封装？
将类的成员变量私有化，隐藏内部实现细节，对外提供公共访问方法，以此保护数据安全。
21
如何使用 JDK 编译与运行 Java 小程序？
编译：执行命令 javac 文件名.java，生成 class 字节码文件。
运行：编写 HTML 文件嵌入小程序代码，执行 appletviewer 文件名.html 运行。
22
Java 语言中什么是构造方法？构造方法有什么作用？
构造方法：方法名与类名完全相同，没有返回值类型的特殊方法。
作用：创建类的对象、初始化对象的成员变量。
23
使用 JDBC 开发数据库应用程序的主要步骤？
加载数据库驱动 → 获取数据库连接 → 创建语句对象 → 执行 SQL 语句 → 处理查询结果 → 关闭相关资源。
